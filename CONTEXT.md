# Label Governance

本上下文描述通过 GitHub Action 校验治理配置并在组织仓库间同步标签的领域语言。

## Language

**Action 调用**：
GitHub Actions 触发的一次完整标签治理任务，模式为离线配置校验、标签同步预览或标签同步写入之一。
_Avoid_：运行（容易与单次仓库同步或内部执行阶段混淆）、调用流程（属于实现描述）

**Action 报告**：
一次 Action 调用无论成功或失败的最终公开结果，涵盖离线配置校验或标签同步，并保持任务摘要、outputs 与完成状态的语义一致；不包含执行过程中的实时日志。
_Avoid_：运行报告（容易排除离线校验）、任务摘要（仅是报告的一部分）
