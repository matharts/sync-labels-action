# 分诊标签

各项技能使用五种规范分诊角色。本文件将这些角色映射到仓库 Issue 跟踪器中实际使用的标签名称。

| `mattpocock/skills` 中的标签 | 本仓库标签        | 含义                            |
| ---------------------------- | ----------------- | ------------------------------- |
| `needs-triage`               | `needs-triage`    | 需要维护者评估此 Issue          |
| `needs-info`                 | `needs-info`      | 等待报告者补充信息              |
| `ready-for-agent`            | `ready-for-agent` | 说明完整，可交由 AFK 智能体处理 |
| `ready-for-human`            | `ready-for-human` | 需要由人工实现                  |
| `wontfix`                    | `wontfix`         | 不会处理                        |

当技能提及某个角色时，例如“应用可由 AFK 智能体处理的分诊标签”，应使用上表中对应的实际标签名称。

如需匹配本仓库已有的标签命名，请编辑表格右侧一列。
